The jsmoke program create, compile and run your java project by using
a simple environment schema and lightweight tooling.

It isn't a successor of widely used programs such as [maven] or
[gradle]. It's actually an alternative for devs that _wan't to try
new things_ and dive deep into _Rust's environment_.

[gradle]: https://gradle.org/
